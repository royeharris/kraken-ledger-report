Use this workflow to commit and push changes to the Kraken Ledger Report. This will trigger a Cloudflare Pages deployment.

1. Navigate to the `kraken-ledger-report` directory:
   `cd /Users/royeharris/Library/CloudStorage/OneDrive-HaricomLimited/Vibecoding/ClawWork/kraken-ledger-report`

2. Check the status of your changes:
   `git status`

3. Stage your changes:
   `git add .`

4. Commit your changes:
   `git commit -m "Update Kraken Ledger Report"`

5. Push to GitHub:
   `git push origin main`

6. Verify the deployment on Cloudflare Pages.
